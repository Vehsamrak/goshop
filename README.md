# Shop abstraction in Go

Package can be used for shop implementation

### Entities list

__Entities__ and their properties.

* __Shop__
	- identificator
	- name
	- description
	* __Location__
		- x
		- y
		- z
		- latitude
		- longitude
		- address
* __Box__
	- identificator
	- name
	* __Pack__
		- identificator
		- quantity
		- discount percent
		* __Merchandise__
			- name
			- price
